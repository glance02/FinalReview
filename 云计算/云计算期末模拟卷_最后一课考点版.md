# 云计算期末模拟卷（最后一课考点版）

> 参考 2025 秋云计算回忆版题型结构命制。闭卷，考试时间 100 分钟，满分 100 分。
>
> 题型：论述题、场景分析题、计算题。无选择题、填空题。请写出必要的概念、判断依据、过程和结果。

## 一、云计算部署模式与服务模式分析（10 分）

某高校计划建设统一信息化平台，包含以下业务：

- 学校官网、招生宣传页面，需要面向公网访问；
- 教务系统、学生成绩、财务数据，只允许校内用户和 VPN 用户访问；
- 期末选课高峰时，希望临时增加计算资源，平时资源自动回收；
- 教师希望直接使用在线文档、会议、邮件等办公软件；
- 信息中心希望租用云主机，自行安装操作系统、数据库和业务系统。

请回答：

1. 该平台适合采用公有云、私有云、混合云、社区云中的哪一种或哪几种部署模式？说明理由。（4 分）
2. 判断上述“在线办公软件”“租用云主机”“开发运行平台”分别属于 IaaS、PaaS、SaaS 中的哪一类，并说明每一类主要由谁负责什么。（4 分）
3. 如果该高校与多所高校共同建设教育资源共享平台，该部分更接近哪一种云？为什么？（2 分）

## 二、CAP、BASE 与一致性场景分析（10 分）

某电商系统有三个机房 A、B、C。用户下单后，订单主库在 A 机房写入成功，但由于网络故障，B、C 机房暂时无法同步最新订单状态。此时用户可能从 B 或 C 机房查询订单。

请回答：

1. 从 CAP 理论角度，网络故障对应哪一个属性？在这种情况下，系统为什么不能同时完美保证一致性和可用性？（3 分）
2. 如果系统要求“查询必须返回最新订单状态，否则拒绝服务”，该系统更接近 CP 还是 AP？牺牲了什么？（2 分）
3. 如果系统要求“先返回旧状态或处理中状态，稍后再自动同步为最新状态”，该系统更接近 CP 还是 AP？它体现了 BASE 理论中的哪些思想？（3 分）
4. 对“订单支付扣款”和“商品评论数量统计”两个业务，分别说明更适合强一致还是最终一致，并说明理由。（2 分）

## 三、一致性散列计算与扩容分析（5 分）

某分布式缓存系统使用一致性散列。哈希环范围为 `0-99`，服务器节点位置如下：

| 服务器 | 哈希环位置 |
| --- | --- |
| S1 | 10 |
| S2 | 40 |
| S3 | 75 |

数据对象的哈希值如下：

| 对象 | 哈希值 |
| --- | --- |
| K1 | 8 |
| K2 | 12 |
| K3 | 38 |
| K4 | 41 |
| K5 | 80 |

规则：对象顺时针找到遇到的第一个服务器作为存储节点。

请回答：

1. 写出扩容前每个对象应存储在哪个服务器上。（2 分）
2. 现在新增服务器 S4，位置为 60。写出新增后每个对象的存储位置。（2 分）
3. 与普通 `hash(key) % N` 相比，一致性散列在扩容时减少数据迁移的原因是什么？（1 分）

## 四、虚拟化技术与轻量级虚拟化（10 分）

某企业计划把传统机房改造成云数据中心，需要同时运行旧版操作系统、Linux 微服务、边缘侧轻量应用和高性能计算任务。

请回答：

1. 说明全虚拟化、半虚拟化、硬件辅助虚拟化的基本思想，并比较它们在兼容性和性能开销上的差异。（4 分）
2. Hypervisor 在服务器虚拟化中承担什么作用？裸金属架构和寄生架构有什么区别？（2 分）
3. Docker 为什么被称为轻量级虚拟化？请从内核共享、隔离机制、资源限制和启动速度角度说明。（2 分）
4. WebAssembly 更适合哪些轻量应用场景？它与 Docker 在运行位置和隔离方式上有什么差别？（2 分）

## 五、云计算网络：覆盖网、大二层、VXLAN 与 GRE（10 分）

某云服务商有两个数据中心 DC1 和 DC2，底层网络已经是三层 IP 网络。现在要求：不同租户之间网络隔离；同一租户的虚拟机可以跨数据中心迁移；迁移后虚拟机 IP 地址和二层网络关系尽量不变；同时希望网络改造成本较低。

请回答：

1. 什么是覆盖网络？为什么该场景适合使用覆盖网络？（2 分）
2. 什么是大二层网络？它和覆盖网络的关系是什么？（2 分）
3. 比较 VLAN 与 VXLAN：分别说明标识空间、跨三层扩展能力、典型适用场景。（3 分）
4. GRE 的基本封装思想是什么？它与 VXLAN 在云数据中心多租户网络中的侧重点有什么不同？（2 分）
5. VXLAN 中 VTEP 和 VNI 分别起什么作用？（1 分）

## 六、SDN 与 NFV 架构分析（10 分）

某云平台希望实现如下能力：网络策略可以由平台统一下发；新增租户网络不需要人工逐台配置交换机；防火墙、负载均衡、VPN 网关可以像虚拟机一样快速创建和迁移。

请回答：

1. 什么是 SDN？请说明控制平面和数据平面分离的含义。（2 分）
2. 画出或文字描述 SDN 的基本架构，包括应用层、控制层、基础设施层、北向接口和南向接口，并说明各部分作用。（4 分）
3. 什么是 NFV？举出三个可以虚拟化的软件网络功能。（2 分）
4. 比较 SDN 与 NFV：它们分别解决什么问题？二者如何配合支撑云网络？（2 分）

## 七、HDFS 文件管理与故障恢复（20 分）

某 Hadoop 集群包含 1 个 NameNode 和 4 个 DataNode：DN1、DN2、DN3、DN4。HDFS 块大小为 128 MB，副本因子为 3。现在有一个 300 MB 文件 `log.dat` 要写入 HDFS。

假设 NameNode 为三个数据块分配的副本位置如下：

| Block | 副本位置 |
| --- | --- |
| B1 | DN1、DN2、DN3 |
| B2 | DN2、DN3、DN4 |
| B3 | DN1、DN3、DN4 |

请回答：

1. `log.dat` 会被切分成几个 Block？每个 Block 的大小分别是多少？（3 分）
2. HDFS 中 NameNode、DataNode、Secondary NameNode 分别负责什么？注意说明真实数据是否经过 NameNode 传输。（4 分）
3. 描述客户端写入该文件到 HDFS 的主要流程，包括向 NameNode 请求、Block 切分、DataNode 管道写入、副本确认和元数据更新。（5 分）
4. 描述客户端读取 `log.dat` 的主要流程。NameNode 返回什么信息？客户端如何读取真实数据？（3 分）
5. 如果 DN3 故障，哪些 Block 的副本数会不足？NameNode 如何发现故障并安排副本恢复？（3 分）
6. HDFS 为什么适合大文件顺序读写，而不适合大量小文件和低延迟随机读写？（2 分）

## 八、MapReduce 计算题（15 分）

某社交平台记录用户关注关系，输入文件每行表示一条关注关系：`关注者 被关注者`。

输入数据如下：

```text
Alice Bob
Alice Carol
Bob Alice
Bob Carol
Carol Alice
David Bob
David Carol
Eve Alice
```

现在使用 MapReduce 统计“每个用户被多少人关注”。假设输入被切成两个 Split：

```text
Split 1:
Alice Bob
Alice Carol
Bob Alice
Bob Carol

Split 2:
Carol Alice
David Bob
David Carol
Eve Alice
```

请回答：

1. 说明该任务中 Mapper 输入的 key/value 可以如何理解。要求写出 `key_P` 和 `value_C` 的含义。（2 分）
2. 对 Split 1 和 Split 2 分别写出 Mapper 输出的所有中间键值对。（4 分）
3. 写出 Shuffle/Sort 后按 key 分组的结果。（3 分）
4. 写出 Reducer 的计算逻辑和最终输出结果。（4 分）
5. 如果要统计“每个用户关注了多少人”，Mapper 输出的 key 应该如何调整？（2 分）

## 九、HBase 数据建模与云存储理解（10 分）

某在线学习平台需要保存学生选课和课程成绩信息。关系型数据表如下：

| S001 | Zhang | Computer |

| StudentID | CourseID | CourseName | Score | Term |

请回答：

1. HBase 为什么适合海量数据的随机实时读写？请结合 Table、RowKey、Column Family、Region、Store、MemStore、StoreFile 等概念说明其基本存储结构。（4 分）
2. 如果查询最常见的需求是“按学生查询该学生全部课程成绩”，请设计一个 HBase 表。说明 RowKey 如何设计，至少给出两个列族及其列名。（3 分）
3. 将上面 `S001` 学生的两条选课成绩样例写成 HBase 表中的两行或一行宽表数据，要求体现 RowKey、列族和列限定符。（2 分）
4. HBase 与传统关系型数据库相比，在事务、Join、扩展性方面有什么典型差异？（1 分）
