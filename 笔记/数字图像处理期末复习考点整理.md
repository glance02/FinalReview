#### 几何变换

1. 平移和缩放，直接对着坐标系做就行
2. 绕原点旋转
$$  
R=  
\begin{bmatrix}  
\cos\theta&-\sin\theta&0\\  
\sin\theta&\cos\theta&0\\  
0&0&1  
\end{bmatrix}  
$$

3. 镜像，也很简单，关于哪个轴镜像就翻转另一个轴的值。关于原点镜像就两个都翻转。关于y=x镜像，就交换x、y的值
4. 错切
	1. x 方向错切
$$
H_x=  
\begin{bmatrix}  
1&k&0\\  
0&1&0\\  
0&0&1  
\end{bmatrix}  
$$
	2. y 方向错切
	$$ 
H_y=  
\begin{bmatrix}  
1&0&0\\  
k&1&0\\  
0&0&1  
\end{bmatrix}  
$$

5. 缩放。形状上的缩放会改变画布大小。[[第3章 图像基本运算-讲到第10页了.pdf#page=39&selection=2,0,6,1&color=yellow|p.39]]
#### 插值算法

1. 最近邻插值。非整数坐标点的灰度值，等于距离它最近的整数像素点的灰度值。
2. 双线性插值。看周围四个像素，根据距离远近做加权平均

#### 算术运算汇总表

| 运算     | 概念                                             | 作用                   | 适用场景                               |
| ------ | ---------------------------------------------- | -------------------- | ---------------------------------- |
| **加法** | 两幅图像对应像素灰度值相加：$g(x,y)=f_1(x,y)+f_2(x,y)$       | 图像合成叠加、多帧平均去噪、拓宽信息   | 多帧平均去噪；图像叠加融合；多光谱加宽波段（绿+红≈全色）；图像拼接 |
| **减法** | 两幅图像对应像素灰度值相减：$g(x,y)=f_1(x,y)-f_2(x,y)$       | 检测两幅图像的差异、去除背景、提取变化  | 运动/变化检测；医学数字减影；去背景、缺陷检测；答题卡识别      |
| **乘法** | 两幅图像对应像素灰度值相乘：$g(x,y)=f_1(x,y)\times f_2(x,y)$ | 提取/屏蔽感兴趣区域、掩膜运算、局部增强 | 掩膜提取 ROI；局部显示；增强对比度                |
| **除法** | 两幅图像对应像素灰度值相除：$g(x,y)=f_1(x,y)/f_2(x,y)$       | 校正乘性干扰、比值运算、归一化      | 校正光照不均（平场校正）；比值运算（如遥感植被指数）         |

> 补充应用题素材：**答题卡识别**多用减法（做差找填涂位置）；**皮肤美白**多用加法/乘法（整体提亮或按系数放大灰度）。

##### DFT特性

| 特性     | 含义                                 |
| ------ | ---------------------------------- |
| 可分性    | 二维DFT可拆成先行后列两次一维DFT                |
| 线性     | $\text{DFT}[af_1+bf_2]=aF_1+bF_2$  |
| 周期性    | DFT结果在频域上是周期性的，周期为N                |
| 共轭对称性  | 实数图像的DFT具有共轭对称：$F(u,v)=F^*(-u,-v)$ |
| 平移特性   | 空域平移对应频域乘相移因子；反之亦然                 |
| 旋转性    | 图像旋转θ角，其频谱也旋转θ角                    |
| 比例变换特性 | 空域压缩对应频域扩展，反之亦然                    |

##### 低频与高频对应的图像信息

| 频率成分   | 对应图像信息                 | 应用           |
| ------ | ---------------------- | ------------ |
| **低频** | 图像整体亮度、大面积平坦区域、缓慢变化的背景 | 低通滤波→图像平滑/去噪 |
| **高频** | 边缘、轮廓、纹理细节、噪声          | 高通滤波→边缘检测/锐化 |

##### 频谱图绘制要点
- 中心化：乘$(-1)^{x+y}$将零频移到频谱图中心
- 中心亮点代表低频（直流分量），越远离中心频率越高
- 通常取对数显示$\log(1+|F(u,v)|)$以压缩动态范围

##### 常见噪声类型与最优滤波方案

| 噪声类型        | 特征                       | 最优滤波方案         |
| ----------- | ------------------------ | -------------- |
| 高斯噪声        | 各像素独立、服从正态分布，整幅图像呈现均匀颗粒感 | 均值滤波、高斯滤波      |
| 椒盐噪声（脉冲噪声）  | 随机出现的纯黑(0)或纯白(255)孤立点    | 中值滤波           |
| 均匀噪声        | 噪声值在某区间内均匀分布             | 均值滤波           |
| 乘性噪声（如斑点噪声） | 噪声强度与像素值成正比，常见于SAR/超声图像  | 同态滤波、对数变换后均值滤波 |

##### 四种空域滤波对比

| 滤波方法 | 原理                    | 优点             | 缺点                 | 适用场景               |
| ---- | --------------------- | -------------- | ------------------ | ------------------ |
| 均值滤波 | 邻域内所有像素取算术平均          | 实现简单，对高斯噪声效果好  | 模糊边缘和细节            | 高斯噪声、均匀噪声          |
| 高斯滤波 | 用高斯加权核做卷积，中心权重大、边缘权重小 | 平滑自然，模糊比均值滤波可控 | 仍会模糊边缘             | 高斯噪声，需要较自然平滑效果     |
| 中值滤波 | 取邻域像素排序后的中值替代中心像素     | 有效去除椒盐噪声，边缘保留好 | 对高斯噪声效果一般；窗口大时细节丢失 | 椒盐噪声/脉冲噪声          |
| 双边滤波 | 同时考虑空间距离权重和灰度相似性权重    | 去噪同时保边效果最好     | 计算量大               | 需要保边去噪（如人脸美化、纹理保留） |

#### 改进算子

1. LOG算子
	- 原理：先用高斯滤波平滑图像去噪，再用拉普拉斯算子求二阶导检测边缘（过零点即为边缘）
	- 作用：解决拉普拉斯算子对噪声极度敏感的问题，兼顾去噪与精确边缘定位
	![[Pasted image 20260704214854.png]]
2. Canny算子
	- 原理（流水线）：高斯平滑→求梯度幅值和方向→非极大值抑制（细化边缘）→双阈值检测→边缘连接
	- 三个最优准则：①低错误率（尽可能少漏检/误检）；②高定位精度（检测到的边缘与真实边缘位置接近）；③单边缘响应（一条真实边缘只产生一个检测响应）
	- 作用：公认最优的边缘检测算子，边缘连续、细而准确

| RGB       | Red, Green, Blue                  | 三基色加色模型，用三原色光叠加表示颜色 | 显示器、摄像机、图像采集与显示                |
| CMY(K)    | Cyan, Magenta, Yellow (Key/Black) | 三补色减色模型，颜料/油墨吸收光    | 彩色印刷、打印                        |
| HSI/HSV   | Hue, Saturation, Intensity/Value  | 色调-饱和度-亮度，符合人类感知方式  | 图像分割、颜色识别、图像检索                 |
| YUV/YCbCr | Y亮度，U/V色度                         | 亮度与色度分离             | 电视广播（PAL/NTSC）、视频压缩（JPEG、MPEG） |

##### 三基色与相加混色

- **三基色**：红(R)、绿(G)、蓝(B)三种颜色无法由其他颜色混合得到
- **相加混色特性**：
  - R + G = 黄(Yellow)
  - R + B = 品红(Magenta)
  - G + B = 青(Cyan)
  - R + G + B = 白(White)
- 彩色图像平滑：对各通道分别做空域滤波（均值/中值/高斯等），或在HSI空间中仅对亮度I通道滤波以保持色彩不失真

##### 退化模型总结

图像退化的一般模型：$g(x,y) = f(x,y) * h(x,y) + n(x,y)$

| 组成部分      | 符号       | 含义                        |
| --------- | -------- | ------------------------- |
| 原始图像      | $f(x,y)$ | 未退化的理想图像                  |
| 退化函数（PSF） | $h(x,y)$ | 点扩散函数，描述模糊/畸变过程（如运动模糊、离焦） |
| 噪声        | $n(x,y)$ | 叠加性噪声（与图像信号独立）            |
| 退化图像      | $g(x,y)$ | 实际观察到的图像                  |

频域表示：$G(u,v) = F(u,v) \cdot H(u,v) + N(u,v)$

- **连续退化模型**：用积分表示，$g(x,y)=\iint h(x,y;a,b)f(a,b)\,da\,db + n(x,y)$
- **离散退化模型**：对连续模型 $f(\alpha,\beta)h(x-\alpha,y-\beta)$ 均匀取样得到，二维离散卷积形式：
$$g_e(x,y)=\sum_{m=0}^{M-1}\sum_{n=0}^{N-1}f_e(m,n)\,h_e(x-m,y-n),\quad x=0\sim M-1,\; y=0\sim N-1$$
  也可用矩阵向量表示：$g = Hf + n$，H 为退化矩阵

> 图像复原的目标：已知退化图像 $g$，估计退化函数 $h$ 和噪声 $n$ 的特性，尽可能恢复原始图像 $f$。

| -------- | ------- | ------------ | ------------- |

#### 典型图像复原方法

| 方法              | 核心思想           | 适合场景           | 优点               | 缺点         |
| --------------- | -------------- | -------------- | ---------------- | ---------- |
| 逆滤波             | 直接除以退化函数 (H)   | 噪声很小，退化函数已知    | 简单，去模糊直接         | 极易放大噪声     |
| 维纳滤波            | 在去模糊和抑制噪声之间折中  | 有模糊也有噪声        | 稳定，常用            | 需要估计功率谱或参数 |
| 等功率谱滤波          | 假设噪声和图像功率谱比例固定 | 功率谱未知，只能粗略估计   | 简单，比逆滤波稳         | 假设粗糙       |
| 几何均值滤波          | 逆滤波和维纳滤波的折中    | 想灵活控制清晰度和噪声    | 参数灵活             | 调参麻烦       |
| 约束最小二乘方滤波       | 加入平滑约束，限制噪声放大  | 噪声能量可估计，希望结果平滑 | 抗噪较好             | 可能损失边缘     |
| Richardson-Lucy | 迭代估计原图         | 光学模糊、泊松噪声      | 去模糊强，适合 PSF 已知场景 | 迭代多会放大噪声   |

## 第十一章 图像分割（纯概念，无计算，可出应用题）
- 各类分割方法原理、适用场景简答

| 方法            | 核心原理                                        | 适用场景              | 主要特点                       |
| ------------- | ------------------------------------------- | ----------------- | -------------------------- |
| **阈值分割**      | 根据灰度值、颜色值是否超过某个阈值，把像素分成目标和背景                | 目标和背景灰度差异明显的图像    | 简单、快，但怕光照不均、噪声             |
| **边界分割**      | 找图像中灰度变化剧烈的位置，也就是边缘，再用边缘围成目标区域              | 物体轮廓清晰、边缘明显的图像    | 擅长找轮廓，但边缘断裂时效果差            |
| **区域分割**      | 根据像素之间的相似性和连通性，把相邻且相似的像素合成一个区域              | 区域内部颜色或灰度比较一致的图像  | 更关注“面”，但容易受噪声、种子点影响        |
| **基于聚类的图像分割** | 把像素看成数据点，根据灰度、颜色、纹理等特征进行聚类                  | 图像中不同类别在特征上有明显差异  | 不一定依赖边界，适合多类别分割，但可能忽略空间连续性 |
| **分水岭分割**     | 把图像看成地形图，灰度或梯度值看成高度，从低洼处开始“灌水”，不同水盆相遇处形成分割线 | 粘连目标分离，例如细胞、颗粒、硬币 | 很适合分开粘在一起的物体，但容易过分割        |
