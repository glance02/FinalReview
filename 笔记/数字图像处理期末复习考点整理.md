#### 位图格式：

| 格式              | 全称/说明               | 常见用途         |
| --------------- | ------------------- | ------------ |
| **BMP**         | Bitmap，Windows 位图格式 | 原始位图、教学示例    |
| **JPEG / JPG**  | 有损压缩格式              | 照片、网页图片      |
| **PNG**         | 无损压缩，支持透明背景         | 网页图标、截图、透明图片 |
| **GIF**         | 支持简单动画，颜色数较少        | 表情包、简单动图     |
| **TIFF / TIF**  | 高质量图像格式，可无损         | 扫描件、印刷、医学图像  |
| **WEBP**        | Google 推出的高压缩格式     | 网页图片         |
| **RAW**         | 相机原始图像数据            | 摄影后期处理       |
| **HEIF / HEIC** | 高效率图像格式             | iPhone照片     |
| **PCX**         | 较早的位图格式             | 旧式图像处理软件     |
| **TGA**         | Targa格式             | 游戏贴图、图形渲染    |
